# Repository Inventory Map

## Folder Structure
* `backend/` - FastAPI web app routes and engines
* `docs/` - Threat model, research documentation, and artifact guides
* `frontend/` - SOC HUD HTML/CSS/JS files
* `ml/` - Training, evaluation scripts, and serialized classifier models
* `scripts/` - Attacker simulation and benchmark run scripts
* `tests/` - Reinforcement learning convergence unit tests
