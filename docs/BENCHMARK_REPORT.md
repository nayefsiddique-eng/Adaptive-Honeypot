# PRAETOR Scientific Benchmark & Evaluation Report

This report presents internal simulation-based benchmarks generated by PRAETOR's own digital-twin framework (synthetic attacker personas evaluated against PRAETOR's own reward function). These results demonstrate internal consistency and relative performance across configurations, but have not been validated against real-world attacker traffic, an independent red team, or a production honeypot deployment. External validation is planned future work.

## 5. Methodology & Limitations
All comparative baselines (Static Honeypot, Random Deception, No Deception, PRAETOR/CMARL) are simulated by the same internal digital twin under the same codebase. The confidence intervals reported reflect variance in the simulator's random seeds and exploration patterns, not the variance of observed real-world attacker behaviors or independent adversary groups.
